SWEDISH MASSAGE PICTURE TRAINER

Contents:
- index.html: complete offline game
- manifest.webmanifest: installable PWA settings
- sw.js: offline cache
- icon.svg: app icon

Fastest use:
1. Open index.html in a browser on a computer.
2. For iPhone/Android installation, upload the whole folder to GitHub Pages or another HTTPS host.
3. Open the website in Safari/Chrome.
4. Choose Add to Home Screen.
5. Open it once while online. After that it works offline.

The single index.html contains all questions, translations, illustrations and game logic. It does not use external images, fonts or libraries.
